## Usage

```bash
omni config schema [OPTIONS] <schema>
```

---

## Positional arguments

### `<schema>`

- **Required:** Yes

- **Possible values:** `workspace`, `project`, `generator`

---

## Options

### `-p`, `--pretty`

Pretty print the schema

- **Required:** No

- **Possible values:** `true`, `false`
