## Usage

```bash
omni hash project [OPTIONS] <project>
```

---

## Positional arguments

### `<project>`

Project name

- **Required:** Yes

---

## Options

### `-t`, `--task`

Hash specific task

- **Required:** No

---

## Examples

```bash
omni hash project [OPTIONS] <project>
```
