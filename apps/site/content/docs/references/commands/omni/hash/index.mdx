## Usage

```bash
omni hash [OPTIONS] <SUBCOMMAND>
```

---

## Options

### `-r`, `--raw-value`

Only output the raw string representation of the hash, no trailing newline, and no traces

- **Required:** No

- **Default:** `false`

- **Possible values:** `true`, `false`

---

## Subcommands

- [`workspace`](./workspace) - Get the hash for the workspace

- [`project`](./project) - Get the hash for a project
