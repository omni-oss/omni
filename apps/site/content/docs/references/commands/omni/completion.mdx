---

title: "completion"

description: "Print shell completions"

---



## Usage

```bash
omni completion [OPTIONS]
```

---

## Options

### `-s`, `--shell`

Which shell to generate completion for

- **Required:** No

- **Default:** `bash`

- **Possible values:** `bash`, `elvish`, `fish`, `powershell`, `zsh`

---

## Examples

```bash
omni completion [OPTIONS]
```
